# نسخة V15 — Safe Costing + Mobile Fix Checklist

هذه النسخة لا تحذف أي بيانات موجودة. التحديث يضيف أعمدة جديدة فقط في آخر الشيتات ويقرأ الأعمدة القديمة للتوافق.

## أهم التعديلات

1. **Waste / الهالك**
   - Sortex Waste = Operation Waste.
   - Caps & Label (PO) = Operation Waste.
   - Line Sludge / طينة الخط = Operation Waste.
   - Bag & Strap = Material Waste.
   - Green Bottle = Material Waste.
   - شاشة Waste أصبحت جدول احترافي مثل شاشة الغلاية مع إجمالي Operation / Material / Total مباشر.

2. **PH**
   - Boiler PH يعمل على مدى 0–100 كما طلبت.
   - Sand Filter و Rinse Tank يعملان على مدى 0–14.
   - لا يتم منع قراءة Boiler العالية، لكن تظهر حالة تنبيه ليتنبه المشغل.

3. **Utilities / المرافق**
   - الكهرباء: استهلاك = نهاية العداد - بداية العداد، ثم تكلفة = الاستهلاك × سعر المصدر/الكاش.
   - المياه: مهيأة على Giza - 6 October / Industrial.
   - الصودا القشور: حساب بالكيلو.
   - الصودا السائلة: حساب باللتر، مع كثافة وتركيز.
   - وقود الفورك لفت: ديزل/بنزين 80/92/95 + لترات + سعر لتر من مصدر رسمي/كاش.

4. **تشغيل / Cost per MT**
   - تكلفة التشغيل/طن = (المرافق + الصودا + وقود الفورك لفت + تكلفة الجواني + العمالة) ÷ الإنتاج بالطن.
   - تكلفة الجواني الكبيرة والصغيرة والشكاير مهيأة للربط من Price Sources.
   - يومية العامل مهيأة للربط من Payroll/Price Sources، ويمكن إدخالها في الفورم لو لم يوجد مصدر.

5. **Price Sources**
   - تم إضافة شيت Price Sources.
   - تم إضافة Tariff Cache حتى لا تتغير الحسابات القديمة إذا تغير السعر لاحقًا.
   - الكهرباء والمياه والصودا والجواني والعمالة والوقود كلهم لهم مصدر وسعر وحالة.

## حل مشكلة الموبايل: Cannot reach Apps Script

هذه الرسالة تظهر غالبًا بسبب أن رابط Apps Script غير منشور Web App بشكل صحيح أو الصلاحيات ليست Anyone.

### خطوات التركيب
1. افتح Google Sheet.
2. Extensions → Apps Script.
3. انسخ محتوى `AppsScript_Code.gs` بالكامل.
4. احفظ الملف.
5. Run → `setupFlakesSystem` مرة واحدة ووافق على الصلاحيات.
6. Deploy → New deployment → Web app.
7. Execute as: Me.
8. Who has access: Anyone.
9. انسخ رابط Web App الجديد.
10. افتح `app.js` وغيّر قيمة `APPS_SCRIPT_URL` إلى الرابط الجديد.
11. ارفع ملفات الداش بورد على Hosting أو افتحها من رابط HTTPS.

## ملاحظات مهمة

- لا يتم استخدام سعر وهمي. لو المصدر لا يعطي API/Regex واضح، يتم حفظ الحالة مثل `NEEDS_SOURCE` بدل حساب رقم غلط.
- أسعار الصودا والجواني والعمالة ليست لها جهة حكومية موحدة مثل الكهرباء والوقود، لذلك يجب ربط مصدر مورد/منصة/ERP أو API.
- كل سجل يحفظ السعر المستخدم والمصدر ووقت التحديث.
